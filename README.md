# MMT.lua  
**Mining Tool for Arizona RP (ПК + Mobile)**

**MMT | Mining Tool** — помощник для майнинга: помогает **собирать прибыль**, **заливать охлаждение**, **включать/выключать стойки**, а также **пополнять банк домов** — всё через удобное окно скрипта.

Работает на **ПК** и **Mobile**.

---

## Основные возможности

### 1) Действия «в 1 клик» по майнингу
- **Сбор прибыли** с полок *(BTC / ASC)*  
- **Заливка охлаждения** на полках  
- **Включение / выключение** полок  
- Работа как:
  - по **одной стойке/полке**  
  - так и через **флешку** со списком домов и полок

Можно собирать всю крипту. Со ВСЕХ домов ОДНОЙ кнопкой!

---

### 2) Пополнение банка домов
- Отдельный режим для **банка домов**: быстрое пополнение без ручной рутины  
- Режимы пополнения:
  - **до заданной суммы**
  - **до максимума** *(19.999.999$)*

---

### 3) Автозаливка охлаждения
Опция автозаливки: при открытии стойки скрипт сам реагирует на низкий процент и заливает охлаждающую жидкость.

**Порог автозаливки** настраивается: **«заливать при X% и ниже»**.

> **Пример (по умолчанию 50%)**  
> Если охлаждение **50% или ниже** — скрипт зальёт **одну** доступную жидкость.  
> Если **50.1% и выше** — заливку выполнять **не будет**.

---

### 4) Чёрный список домов
Можно скрыть (исключить) дома, которые не нужно трогать *(например: пустые / чужие)*.

---

### 5) Замена стандартных диалогов на окно скрипта
Скрипт может заменять игровые диалоги майнинга на **своё окно** *(отключается в настройках)*.

Полезно на **мобильных**, чтобы диалоги не занимали лишнее место на экране.

---

### 6) Раздел «Улучшить» (заточка/улучшение видеокарт)
Встроенный помощник для улучшения видеокарт:

- Улучшение **всех карт** или **выбранной**  
- Порядок улучшения:
  - **последовательно**
  - **как на экране**
- Ограничение по **целевому уровню** *(не улучшать выше заданного)*  
- Опциональная проверка наличия **«смазки»** перед стартом  
- Вкладки: **Процесс / Настройки / Логи** *(ведётся статистика и логирование)*

На данный момент работает только со старым инвентарем. Работаю над переносом на новый CEF.

# Как пользоваться

## Быстрый старт
1. Установите скрипт в папку:
   - **ПК:** MoonLoader  
   - **Mobile:** MonetLoader
2. Зайдите в игру и откройте окно командой:  
   - **`/mmt`** — открыть/закрыть главное окно MMT
3. Если окно появилось — скрипт установлен и работает ✅  
4. Откройте **флешку** или **стройку с видеокартами** и пользуйтесь функциями MMT.

---

## Команды
- **`/mmt`** — открыть/закрыть окно  
- **`/mmtr`** — перезагрузить скрипт  
- **`/mmtsr`** — сбросить масштаб интерфейса к `1.0` и перезагрузить скрипт  

---

# Настройки

## Основное
- **Заменять диалог на окно скрипта** — включает/выключает кастомное окно вместо стандартного диалога  
- **Закрывать на ESC** — удобно, если играете на ПК  
- **Перемещение стрелками по спискам** — работает на ПК  
- **Скрыть текст получения крипты в чат** — меньше спама  
- **Панель статуса (debug)** — показывает активные процессы *(нужно только для отладки)*

---

## Порог заливки
- **«Заливать при X% и ниже»** — главный параметр для заливки/автозаливки

---

## Доходность (отображение)
Можно включить/выключить показ:
- доход **/ час**
- доход **/ 24ч**
- доход **/ цикл**
- расчёт **«до доливки»** *(время/прибыль)*

---

## Интерфейс
- **Масштаб UI** — насколько крупным/мелким будет текст и элементы окна  
  - Быстрый сброс: **`/mmtsr`**
- **Размер окна** — окно можно растягивать мышкой за края, размер сохраняется  
  - ⚠️ *Не путать с масштабом UI*
