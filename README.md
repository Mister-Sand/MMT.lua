# MMT.lua
**Mining Tool for Arizona RP (ПК + Mobile)**

**MMT | Mining Tool** — помощник для майнинга на Arizona RP. Скрипт упрощает работу с полками и домами: помогает **собирать крипту**, **заливать охлаждение**, **включать/выключать стойки**, **работать через флешку**, **пополнять банк дома**, а также **улучшать видеокарты** через удобное окно.

Актуальная версия: **1.7**.

Работает на **ПК** и **Mobile**.

---

## Основные возможности

### 1) Сбор, заливка и управление стойками
- **Сбор прибыли** с полок *(BTC / ASC / ASIC)*
- **Заливка охлаждения**
- **Включение / выключение** видеокарт
- Работа:
  - по **одной полке / стойке**
  - по **дому целиком**
  - через **флешку** со списком домов

Можно быстро собрать крипту сразу со всех доступных домов одной кнопкой или командой.

---

### 2) Автосбор через флешку
Добавлен отдельный сценарий сбора через флешку:
- кнопка **«Открыть флешку и собрать»** в главном окне;
- команда **`/mmtflash`** для быстрого запуска;
- можно отменить процесс прямо из окна скрипта.

Это удобно, когда нужно пройти весь маршрут домов без ручного открытия каждого дома по отдельности.

---

### 3) Автозаливка и автовключение видеокарт
Скрипт умеет автоматически помогать после открытия полки:
- **автозаливка** охлаждения при падении процента до заданного порога;
- **автовключение видеокарт после сбора** крипты;
- наглядные переключатели прямо в основном окне.

**Порог автозаливки** настраивается: **«заливать при X% и ниже»**.

> Пример: если установлено **50%**, то при **50% и ниже** скрипт будет заливать охлаждение, а при значении выше — нет.

---

### 4) Доходность видеокарт и бонусы домов
Скрипт показывает расширенную информацию по доходности:
- доход **за час**;
- доход **за 24 часа**;
- доход **за цикл**;
- **текущую прибыль**;
- **время до доливки**.

В версии **1.7** добавлен **подсчёт прибыли с учётом бонусов дома**. Для домов можно настраивать бонусы доходности, чтобы расчёты совпадали с реальной отдачей.

---

### 5) Логи сбора крипты
Появился отдельный раздел **«Логи»**:
- история сбора по дням;
- детализация по каждому дому;
- отображение суммарного количества **BTC / ASC** за день;
- быстрый просмотр последнего зафиксированного сбора.

Это полезно для контроля фарма и проверки, где именно собиралась крипта.

---

### 6) Напоминания о сборе крипты
Скрипт умеет напоминать, если сбор давно не выполнялся:
- настраивается интервал в **минутах**;
- можно использовать систему **session_notifications**;
- уведомление может содержать кнопку **«Запустить сбор»**.

Если библиотека/менеджер уведомлений не установлены, скрипт покажет статус и предложит подготовить систему уведомлений.

---

### 7) Пополнение банка домов
Есть отдельный режим для банка дома:
- пополнение **до заданной суммы**;
- пополнение **до максимума**;
- отображение текущего баланса по домам.

Подходит для быстрой рутины без ручного ввода сумм в каждом доме.

---

### 8) Чёрный список домов
Можно скрыть или исключить дома, которые не нужно обрабатывать:
- пустые;
- чужие;
- временно неактуальные.

Список сохраняется в настройках скрипта.

---

### 9) Замена игровых диалогов на окно скрипта
Скрипт умеет заменять стандартные диалоги майнинга на **своё окно**.

Это особенно удобно на **мобильных устройствах**, где стандартные диалоги занимают много места.

Опция отключается в настройках.

---

### 10) Улучшение видеокарт
Встроен отдельный раздел **«Улучшить»** для заточки видеокарт:
- улучшение **всех карт** или **одной выбранной**;
- режимы улучшения:
  - **последовательно**;
  - **как на экране**;
- ограничение по **целевому уровню**;
- проверка наличия **смазки** перед стартом;
- отдельные вкладки **Процесс / Настройки / Логи**;
- ведётся логирование результатов.

Дополнительно:
- поддержан **новый стиль инвентаря** для улучшения видеокарт *(ПК)*;
- классический режим тоже сохранён.

## Как пользоваться

## Быстрый старт
1. Установите скрипт в папку:
   - **ПК:** `moonloader`
   - **Mobile:** `monetloader`
2. Зайдите в игру и откройте окно командой **`/mmt`**.
3. Откройте флешку, дом или стойку с видеокартами.
4. Выберите нужный режим: сбор, заливка, банк дома или улучшение карт.

---

## Команды
- **`/mmt`** — открыть/закрыть главное окно
- **`/mmtr`** — перезагрузить скрипт
- **`/mmtsr`** — сбросить масштаб интерфейса к `1.0` и перезагрузить скрипт
- **`/mmtflash`** — запустить сбор через флешку

---
